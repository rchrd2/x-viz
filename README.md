# x-viz

Web component for rendering a graphviz graph using [viz-js](https://github.com/mdaines/viz.js/).

Part of the [x-gui](https://github.com/rchrd2/x-gui) project.

## Example usage

```
<x-viz value="digraph g { a -> b; }"></x-viz>
```
